# Blue Sphere Archipelago

This is an [Archipelago](https://archipelago.miraheze.org/wiki/Archipelago) hint game implementation for [Blue Sphere](https://info.sonicretro.org/Blue_Sphere).
Completing Special Stages grants location hints.

## How to play

Players can refer to the [game page](</docs/en_Blue Sphere.md>), then the [setup guide](/docs/setup_en.md).
